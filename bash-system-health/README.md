# Bash System Health Automation Script

## Overview

This project focused on developing a Bash automation script to generate Linux system health reports automatically.

The script collects system information, reviews service status, identifies open ports, checks authentication log activity, and produces timestamped reports for administrative review.

## Environment

* Ubuntu 26.04 Virtual Machine
* Bash Shell

## Features

* Hostname collection
* IP address reporting
* System uptime reporting
* Disk utilization reporting
* Memory utilization reporting
* CPU information reporting
* Active user identification
* SSH service status checks
* Open port identification
* Authentication log review
* Timestamped report generation

## Skills Demonstrated

* Bash scripting
* Linux automation
* Error handling
* Privilege validation
* System monitoring
* Report generation
* Log review

## Files

* `system_health.sh` – Bash automation script
* `health_report_*.txt` – Sample generated report
* `bash_system_health_automation.pdf` – Complete project documentation

## Key Takeaways

This project strengthened my understanding of Linux automation, Bash scripting fundamentals, error handling, report generation, and administrative monitoring workflows.
