# Linux System Administration & Network Analysis Lab

## Overview

This project was completed to develop practical Linux administration and network analysis skills within an Ubuntu virtual machine environment.

The lab focused on system administration tasks, SSH configuration, user and group management, file permissions, authentication log review, network troubleshooting, and packet analysis using Wireshark.

## Environment

* Ubuntu 26.04 Virtual Machine
* VirtualBox
* OpenSSH Server
* Wireshark

## Skills Demonstrated

### Linux Administration

* User and group management
* File ownership and permissions
* SSH installation and configuration
* Service management using systemctl
* Authentication log review
* Network troubleshooting

### Network Analysis

* DNS query analysis
* TCP handshake analysis
* TLS/HTTPS traffic analysis
* SSH traffic identification
* ICMP traffic analysis
* Packet filtering and protocol investigation

## Files

* `linux_system_administration_network_analysis.pdf` – Complete project documentation
* `wireshark_lab_capture.pcapng` – Packet capture used during network analysis

## Key Takeaways

This project strengthened my understanding of Linux administration, network troubleshooting, authentication logging, and packet-level network analysis using Wireshark.
